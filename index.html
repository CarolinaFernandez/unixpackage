UnixPackage

Stand-alone tool to easily create Debian and Fedora based UNIX packages through a UI

View project on GitHub

Overview

This tool provides an easier way to bundle software into packages for UNIX Operating Systems (currently Debian and Fedora based distros are supported). Further instructions on use can be checked in the GitHub wiki and in the README files available per UNIX operating system.
